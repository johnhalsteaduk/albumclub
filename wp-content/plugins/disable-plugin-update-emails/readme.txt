Disables the default notification email sent by WordPress for automatic plugin updates, regardless of outcome (success or failure). Simply activate the plugin to disable the notification email :)

== Description ==
As of WordPress 5.5, email notifications will be sent after each attempt to automatically update a plugin, regardless of whether the update was successful or failed. To disable these email notifications, simply install and activate this plugin :)

== Installation ==
1. Extract `disable-plugin-update-email.zip` to `/wp-content/plugins/` directory2. Activate the plugin through the 'Plugins' menu in WordPress
3. The plugin works immediately after activation with no additional configuration required.

== Frequently Asked Questions ==
= Does this work on all versions of WordPress? =
No, this plugin will only work on WordPress 5.5 or above.= When would I use this plugin? =This plugin can be used to suppress notifications on your hosted pre-production environments or to disable notifications for your customers' production websites when alternative plugin management tools are utilized.== Changelog ==
